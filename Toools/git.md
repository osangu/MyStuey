`$ git switch {branch-name}`
+ just checkout to {branch-name} 


`$ git switch -c {branch-name}`  

+ create branch and checkout, which name is {branch-name}

`$ git resotre {file-name}`
+ change {file-name} status same as before modified 

`$ git restore -S {file-name}` or `$ git restore --s {file-name}`
+ unstage file by git add command
