### SRP
- 단일 책임 지향 원칙
- 하나의 클래스(모듈, 메서드)는 하나의 역할만 수행해야 한다.

예를 들어, UserRegisterService라는 클래스에서 회원가입을 실행하는 함수 execute의 로직은 아래와 같다.
1. ID & PW 길이 확인
2. ID 중복 여부 확인
3. 저장  

이 때, 위의 모든 과정들을 execute 함수내에서 구현하면 안 된다.
